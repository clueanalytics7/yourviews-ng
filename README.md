# YourViews NG 🇳🇬
**Empowering Citizens Through Data-Driven Policy Insight**

YourViews is a digital community sentiment portal designed to bridge the gap between Nigerian citizens and local policy makers. By providing an anonymous, NDPR-compliant platform for feedback, we turn public opinion into actionable research data.

### 🚀 Key Features
- **Real-time Sentiment Tracking:** Visualize community agreement on local infrastructure, economy, and social projects.
- **Demographic Insights:** Anonymous breakdown of views by age, location, and education level.
- **NDPR Compliant:** Privacy-first design using device fingerprints instead of PII.
- **Bilingual Interface:** Supports English and Pidgin English.

### 🛠 Technology Stack
- **Frontend:** HTML5, Tailwind CSS (Mobile-responsive)
- **Backend:** Google Apps Script (Serverless API)
- **Database:** Google Sheets (Real-time collaborative data)
- **Hosting:** GitHub Pages
